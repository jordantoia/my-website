# Saint Creative Lead Pitch — Project Brief

## Project Overview

Create a professional one-page pitch document (HTML & PDF) for a creative director role at Saint, a Melbourne-based clothing brand. The document must be factual, points-based, and designed to demonstrate value and efficiency rather than advertising/selling.

**Deliverables:**
- Single-page HTML document (A4 format, 210mm × 297mm)

## Brand Context

**Saint** is a Melbourne clothing company with two distinct brand lines:

1. **Saint Clothing** — Durable motorcycle apparel (jeans, jackets) designed for safety. Non-wear-tear fabrics tested for accidents. Target: late 20s to 60s, mature motorcycle riders.

2. **Saint Layers** — Activewear targeting a broader, younger demographic. Growth-stage brand expanding market reach.

**Current State:**

- One in-house editor (~20 hrs/week)
- External Brisbane/Sydney team for basic edits
- Existing social media accounts
- **Lacking:** dedicated shooter/director, planned campaigns, unified creative direction, strategic posting

## Implementation Notes

### Critical Design Decisions

1. **Orange accent color** — Must be applied to:
   - Left page border (4px solid)
   - Header bottom border
   - Section label underlines
   - Day name underlines
   - Block title underlines
   - Footer top border
   - Reference links

2. **Typography hierarchy** — Must be clearly defined:
   - All headings use condensed sans-serif
   - All body text uses regular sans-serif
   - Section labels: 11px bold uppercase with wide letter-spacing
   - Block/brand titles: 15-18px bold uppercase with orange underline
   - Body bullets: 12px regular with custom bullet markers

3. **Grid precision** — Columns must be exact:
   - Equal two-column: `1fr 1fr` with 10px gap
   - Four-column days: `repeat(4, 1fr)` with no gap, borders between
   - Asymmetric lower: `1.1fr 0.9fr` with 10px gap

4. **Spacing system:**
   - Section spacing: 24px between major sections
   - Card padding: 14-16px internal
   - List item spacing: 2-3px vertical padding
   - Line height: 1.5-1.55 for readability

### Print Considerations
- No page breaks (single page document)
- Colors must be print-safe (no pure black #000, use #1a1a1a)
- Orange must be consistent across all instances
- Logo must remain crisp at print resolution
- Links should remain clickable in PDF if possible
